# Apple Touch Icon Placeholder

This file is a placeholder. To add Apple Touch Icons:

1. Create PNG images in the following sizes:
   - 180x180 (required for iOS 7+)
   - 152x152 (iPad)
   - 144x144 (iPad)
   - 120x120 (iPhone)
   - 114x114 (iPhone)
   - 76x76 (iPad)
   - 72x72 (iPad)
   - 60x60 (iPhone)
   - 57x57 (iPhone)

2. Name them:
   - apple-touch-icon.png (180x180 - default)
   - apple-touch-icon-152x152.png
   - apple-touch-icon-144x144.png
   - apple-touch-icon-120x120.png
   - apple-touch-icon-114x114.png
   - apple-touch-icon-76x76.png
   - apple-touch-icon-72x72.png
   - apple-touch-icon-60x60.png
   - apple-touch-icon-57x57.png

3. Place all files in the `/public` directory

4. The icons should be square PNG images with your logo/branding

For now, the HTML includes the proper link tags. When you add the actual icon files, they will automatically be used.
